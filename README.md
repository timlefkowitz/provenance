# Provenance - Art Authentication Platform

## Development

- **This app** lives at the repository root (`src/`, `pnpm dev`). That is the Next.js app you should run and deploy.
- Under `makerkit/nextjs-saas-starter-kit-lite/apps/provenance/` there is a **historical / kit mirror** of an app layout. Do not confuse it with the root app; run a single dev server from the repo root unless you intentionally work inside the Makerkit workspace.
